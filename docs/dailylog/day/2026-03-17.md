- 修复计划事件订阅鉴权方式：编排 SSE 从 `EventSource` 切换为 `fetch` 流式解析，补齐 `Authorization` 请求头并保留 query token 兜底。
- 补充 fix 落盘文档与相关功能/开发文档更新，确保变更可追溯。

- 新增 agents runtime 清理脚本：支持清理 `agent_sessions/agent_messages/agent_parts` 及对应 Redis `agent-task` 队列/事件 key，默认 dry-run，实删需显式确认。
- 重构 `ExecutorSelectionService`：从关键词匹配升级为多维能力路由（角色/工具/能力/关键词），并支持 `lock_to_planner` 分配策略。
- 修复 Orchestration 分配链路：计划创建与 replan 都传入 `plannerAgentId + assignmentPolicy`，支持强约束场景下任务归属锁定。
