# 里程碑

当前我们项目已经实现了agent集群协作和任务规划等主要能力，并且对agent执行过程有完整日志和评分体系的支持。

接下来我们主要需要实现的能力有以下

1. 项目化孵化
2. 飞书接入

## 功能需求

### 项目孵化

- 目标：实现项目化孵化能力，支持用户围绕特定项目创建专属的agent集群，进行持续的任务规划和执行。

目前主要可能孵化以下几种项目类型：
1. WEB3数据收集分析，通过部分网站抓取数据，分析WEB3行业趋势
2. 竞品分析，通过抓取和分析竞品信息，持续输出竞品分析报告
3. 纯研发项目，由agent协助进行技术调研、方案设计、代码生成和测试验证等

### 飞书接入
- 目标：实现飞书接入能力，支持用户通过飞书消息与agent进行交互，接收agent的任务执行结果和日志输出。
